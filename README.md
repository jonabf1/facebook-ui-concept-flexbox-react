#### Como instalar
- $ git clone https://github.com/jonabf1/facebook-ui-concept-flexbox-react
- Vá para a pasta do repositorio
- yarn install (instalar todas as dependências)
- yarn start (iniciar a aplicação)

#### Dependência usadas
- axios
- react
- dotEnv
- react-router-dom
- react-icons
- socket.io
- socket.io-client
- styled-components
