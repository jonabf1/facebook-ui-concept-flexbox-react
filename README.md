<h1 style="text-align:"center";">Facebook-Flexbox-React</h1>

### Como instalar
# Clonar esse repositório
$ git clone https://github.com/jonabf1/facebook-ui-concept-flexbox-react

# Vá para a pasta do repositorio

# Instalar dependências
$ yarn install

# Abrir o app
$ yarn start
